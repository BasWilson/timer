# timer

The single most basic timer tool in existance.

## Install
```bash
pip3 install git+https://github.com/BasWilson/timer.git
```

## Usage
```bash
# Start timer
timer start <project-name>

# Stop timer
timer stop <project-name>

# List active timers
timer list

# Delete all projects
timer delete-all

# Delete single timer
timer delete <project-name>

# Get totals for all projects
timer total
```
